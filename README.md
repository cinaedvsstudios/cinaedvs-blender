# CINAEDVS Blender Support

This public repository is the official bug tracker for CINAEDVS Blender add-ons, including **CINAEDVS File Rescue**.

The add-on source code is not hosted here. This repository exists so users can report reproducible problems, attach screenshots, and follow the status of fixes.

## Report a bug

Use the **Issues** tab and select **Bug report**.

Before submitting, please include:

- Blender version
- Add-on version
- Windows version
- A clear description of what happened
- Steps to reproduce the problem
- Any error message or Blender console output
- Screenshots when useful

Please do not upload private `.blend` files or personal file paths unless you are comfortable making them public.

## Support scope

This tracker is for bugs and compatibility problems in CINAEDVS Blender add-ons. General Blender questions and feature requests may be closed or redirected.
